== Description ==

Elevate user engagement with the inclusion of a floating WhatsApp icon in your website's footer.

## EASY CUSTOMIZATION

SEND ZAP seamlessly integrates into WordPress settings, allowing you to customize all aspects and witness instant results.

## UNIVERSAL FUNCTIONALITY

Sendzap is compatible with all major browsers, ensuring usability across any theme or device.

## SEND ZAP HIGHLIGHTS

- Implement a floating WhatsApp icon in your website's footer, effortlessly configuring the phone number and icon style to suit your preferences.


== Installation ==

1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.

== Frequently Asked Questions ==

= How can I change the settings of the preloader =

Navigate to Settings > SEND ZAP. Here you find the panel with all the SEND ZAP settings.

= How to uninstall the plugin? =

Simply deactivate and delete the plugin.
